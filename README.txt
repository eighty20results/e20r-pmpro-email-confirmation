== Description ==

Provides the `[e20r_confirmation_form]` shortcode. This shortcode can be used to let users re-send the PMPro Email Confirmation message to the specified email address or to the logged in user. 

== Installation ==

1. Upload the `e20r-pmpro-email-confirmation` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
